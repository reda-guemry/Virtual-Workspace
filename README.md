**WorkSphere** – Gestion des employés dans le bâtiment
Description

WorkSphere est une application web qui permet de gérer visuellement et interactivement les employés dans un bâtiment.
Elle affiche le plan du bâtiment avec différentes zones et garantit que les employés sont placés selon leurs rôles et permissions.

**Fonctionnalités** principales

- Ajouter, modifier et supprimer des employés facilement.

- Application automatique des règles d’accès selon le rôle.

- Visualisation en temps réel de la position des employés.

- Profil complet pour chaque employé avec photo et expériences professionnelles.

- Interface responsive adaptée à tous les appareils.

- Les zones vides obligatoires apparaissent en rouge.

**Technologies utilisées**

- HTML5

- CSS3 (Flexbox / Grid)

- JavaScript (ES6+)

- Regex pour la validation des données


- Responsive Design

**Utilisation**

- Ajouter un nouvel employé via le bouton "Add New Worker".

- Cliquer sur un employé pour voir son profil complet.

- Les zones vides importantes sont signalées en rouge.
